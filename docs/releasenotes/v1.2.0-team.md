# Release Notes — v1.2.0 (Team Version)

**Date:** March 12, 2026

**Status:** ✅ COMPLETE

**Audience:** Non-technical team members

**Scope:** Terminology update and small UX improvements

---

## What's New

### "Gigs" is now "Opportunities"

Across the platform, we've updated the wording from **"Gigs"** to **"Opportunities"**. You'll see this change in:

- Navigation menus
- Dashboard headings
- Application and booking screens
- Forms and filters

**Why?** "Opportunities" better reflects the range of work available (modeling, events, creative projects, and more). The change is labels only — your bookmarks and links still work the same.

### More ways to filter

When browsing or creating opportunities, you can now filter by **"Events"** and **"Others"** in addition to the existing types. The filter label has been updated from "Category" to **"Opportunity Type"** for clarity.

### Better image guidance for opportunity flyers

When uploading a cover image for an opportunity, you'll now see a notice with recommended dimensions (1200×900 px, 4:3 ratio). This helps flyers and promotional images display correctly without stretching or cropping.
